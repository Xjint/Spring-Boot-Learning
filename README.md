## 4
* 完善显示问题的页面
* 由于每次修改页面或者改动代码时，都要手动重启服务器，
因此引入“热部署”,引入spring-boot-devtools依赖，以后每次做了改动之后，不需要重启服务器，
只需rebuild项目就好啦。但是，rebuild项目也是得我们手动点击，
所以，在Idea中开启自动编译的选项，这样，我们只要改动了代码，IDE就会自动为我们编译了，我们不需要做任何额外的操作
